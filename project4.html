Project 4 – Server

Installatie en beheer van Windows- en Linux-servers met basisrollen en gebruikersbeheer.

Over het project

In dit project heb ik een server opgezet en ingericht met een basisconfiguratie. Het doel was om te leren hoe je een serverbesturingssysteem installeert en hoe je daarna de belangrijkste instellingen en rollen goed configureert.

Ik heb gewerkt met zowel Windows Server als Linux. Ik leerde hoe je gebruikers en groepen aanmaakt, rechten instelt en mappen deelt binnen het netwerk. Ook heb ik gekeken naar het belang van updates en het veilig instellen van beheeraccounts.

Daarnaast heb ik eenvoudige diensten geconfigureerd, zoals het delen van bestanden en het instellen van remote beheer. Dit liet zien hoe een server gebruikt kan worden als centrale plek binnen een netwerk waar meerdere gebruikers mee werken.

Dit project gaf mij een eerste goed beeld van wat er komt kijken bij serverbeheer. Ik begrijp nu beter welke taken een systeembeheerder heeft en hoe belangrijk stabiliteit en veiligheid zijn bij servers.
